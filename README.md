# Osoi News (遅いニュース)

## Initialisation 

- Ensure Docker Desktop is installed and running
- Run `docker-compose up --build` inside the project directory
- Go to `http://localhost:3000`

## Closing the project

- Run `docker-compose down` and close the browser tab

## Restarting the project

- Run `docker-compose up` and visit `http://localhost:3000`
